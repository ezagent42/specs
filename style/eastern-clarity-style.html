切换行为

html[data-theme="dark"] { --bg:#1e2128; --bg-alt:#15181e; ... } /* 跟随系统 */ @media (prefers-color-scheme: dark) { html:not([data-theme="light"]) { /* dark tokens */ } }
优先级：用户手动 > 系统偏好 > 默认 light。data-theme 存 localStorage，<head> 内联 script 提前设置避免 FOUC。
